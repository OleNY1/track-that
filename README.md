# Track That
Track That is a full-stack web application for tracking internship and job applications in one place.
--------------------------------------------------------------------------------------------------------------------|
## Tech Stack

### Frontend
- HTML, CSS, JavaScript
- React (Vite)
- Tailwind CSS

### Backend
- Python
- FastAPI

### Database
- PostgreSQL  
- SQLite (local development)

### Tools
- VS Code
- Git & GitHub
--------------------------------------------------------------------------------------------------------------------|
## Features (MVP)

- Add a job application
- View all applications
- Update application status
- Delete an application
--------------------------------------------------------------------------------------------------------------------|
## Data model definition

- id
- company
- role
- location
- application_link
- date_applied
- status
- notes
--------------------------------------------------------------------------------------------------------------------|
## Mental model

- backend/ → FastAPI, Python, database logic
- frontend/ → React, UI, forms, tables
